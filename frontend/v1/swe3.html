Brake Controller

Software Detailed Design Specification

1. Introduction

The Brake Controller unit implements the core logic for the Electronic Braking System (EBS). It processes wheel speed sensor data and driver brake requests to modulate hydraulic valve actuators via high-frequency PWM signals.

This unit is allocated to the APP_LAYER / Chassis_Mgmt group and interfaces with the HAL layer through standardised driver abstractions. The unit executes within a 5 ms cyclic task context under the AUTOSAR OS.

2. Interfaces

Input and output interfaces as derived from the SWE.2 Software Architecture.

Interface ID Dir Type Description
IF_BrakeReq Input float32 Target deceleration from VCU [m/s²]
IF_WheelSpeed[4] Input uint16[4] Raw encoder pulse counts per wheel
IF_Pressure_Est Input float32 Estimated hydraulic line pressure [bar]
IF_ValveCmd Output uint8[4] PWM duty cycle for solenoid valves [0–100%]
IF_DiagStatus Output uint32 Bitmask of active diagnostic trouble codes
